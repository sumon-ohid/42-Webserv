Projects

Philosophers

Philosophers is a project where you have to solve the dining philosophers problem using threads and mutexes. The dining philosophers problem is a classic synchronization problem in computer science where a number of philosophers sit at a table and eat spaghetti. The philosophers can only eat if they have both forks and they can only pick up one fork at a time. The challenge is to write a program that allows the philosophers to eat without causing a deadlock.

Philosophers is my favorite project at 42 in C. I really liked digging in to it and thinking about a logic when multiple processes run in paralell and how to make sure that in each case of error all processes are stopped and everything is freed properly
